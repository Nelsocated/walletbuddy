WalletBuddy is a budget tracking application that can help anyone track their finances. This app was supposed to be school project but I recreated it myself again, built it from scratch.

This app is a fullstack application.
Stacks:

- - Frontend - NextJs, TailwindCSS, & Typescript
- - Backend - NestJs, & PrismaORM
- - Database - PostGresSQL
- - Deployment:
    - - Frontend - Vercel
    - - Backend - Docker
