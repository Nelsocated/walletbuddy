# **WalletBuddy**

### **Overview**

A full-stack budgeting application that can help anyone track their finances. This app was supposed to be school project but I recreated it myself again, built it from scratch.

### **Features**

- User authentication
- Create, edit, and delete transactions
- Set budget for the week or month
- Categorize expenses

### **Tech Stacks**

Frontend - NextJs, - TailwindCSS, & - Typescript
Backend - NestJs, - PrismaORM, & - PostGresSQL
